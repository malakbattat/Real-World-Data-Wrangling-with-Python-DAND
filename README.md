# Data Wrangling Project

## Overview

This project focuses on applying data wrangling techniques using Python to collect, assess, clean, and analyze real-world datasets. I will be working on gathering two related datasets from different sources, identifying quality and tidiness issues, and transforming them into a clean and structured format.

## Steps  

- Gather two related datasets from different sources  
- Assess data quality and tidiness issues  
- Clean and restructure the data  
- Store the cleaned data  
- Perform exploratory data analysis (EDA) and visualize insights  

## Status  

This project is currently in progress, and I will be working on it.

## Goal  

To demonstrate practical skills in data wrangling, data cleaning, and exploratory data analysis using Python and its data-focused libraries.
